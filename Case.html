Switch/Case

Escolhe entre várias opções.

Código:

let dia = 3;
switch (dia) {
case 1: console.log("Domingo"); break;
case 2: console.log("Segunda"); break;
default: console.log("Outro dia");
}

Prático:

Mostrar o nome do dia da semana.
